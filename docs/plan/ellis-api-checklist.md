| # | 질문 | 답변 |
|---|------|------|
| 1-1 | 호텔 **검색/콘텐츠/요금 API**의 스펙 문서(엔드포인트, 인증, 요청/응답 예시)가 있습니까? 위치/담당자는? | |

| 1-4 | 지역 코드 체계(예: `102911 - Tokyo`)의 마스터 목록 또는 조회 API가 있습니까? | |
| 1-5 | 요금 조회 API의 **호출당 평균/최대 응답 시간**과 **허용 QPS(초당 호출 수)** 는 어느 정도입니까? AI 검색이 추가로 발생시킬 부하 한도를 지정해 주십시오. | |
| 1-6 | 테스트/스테이징 환경과 테스트 계정이 있습니까? | |

## B-3. 인증·세션 (Orchestrator 인증 설계 확정)

| # | 질문 | 답변 |
|---|------|------|

| 3-2 | 토큰에서 **셀러(Agent) ID·마켓·기준 통화·권한**을 조회할 수 있습니까? 어떤 API로? | |
